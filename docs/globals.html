    A base class can be provided which will be extended including the mixin.

logger

  • logger(key: string): PropertyDecorator
  • logger<T>(target: T, key: string): void
    • Defined in src/LoggerDecorator.ts:3

    Property decorator that will assign a tagged Logger proxy to the decorated property where the tag is the name of the parent class of the property

    Example:

    class FooClass {
    @logger // Logging methods will be tagged with `FooClass`
    private readonly logger: typeof Logger;
    ...

    The decorator can be given a tag string to manually assign a different tag instead

    Example:

    class FooClass {
    @logger('Bar') // Logging methods will be tagged with `Bar`
    private readonly logger: typeof Logger;
    ...

    Note: This is a Typescript feature. If using the logger is desired in Javascript just call the static logging methods on Logger itself Logger.instance()
